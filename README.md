# README

## 開発言語
---
- Ruby2.6.5
- Rails5.2.5

## 就職Termの技術
---
- devise
- rails_admin
- cancancan
- Ajaxを使ったコメント機能
- kaminari

## カリキュラム外の技術
---
- Active Storage
- ransack

## 実行手順
---
```
$ git clone git@github.com:YuiMasuko/continue.git
$ cd continue
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
